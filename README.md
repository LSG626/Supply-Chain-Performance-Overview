# Supply-Chain-Performance-Overview

A comprehensive Power BI dashboard for analyzing and monitoring supply chain performance metrics across multiple locations, suppliers, and product categories.

## Overview

This dashboard provides real-time insights into supply chain operations, helping stakeholders make data-driven decisions through interactive visualizations of key performance indicators (KPIs), cost analysis, shipping performance, and inventory management.

## Key Features

### Performance Metrics
- **Total Revenue:** $102.60K tracked across all operations
- **Total Costs:** $10.56K comprehensive cost monitoring
- **Profit Analysis:** $92.04K profit with 0.90 profit margin percentage
- **Order Volume:** 5,420 total orders processed
- **Defect Rate:** 0.00% quality performance tracking

### Interactive Visualizations

#### 1. Location Analysis
Multi-location selector featuring:
- Bangalore

- Delhi
- Kolkata
- Mumbai

#### 2. Revenue & Cost Analysis
- Side-by-side comparison of total costs vs. total revenue by product type
- Product categories: Haircare, Skincare, Cosmetics

#### 3. Supplier Performance Metrics
Comprehensive supplier analysis table including:
- Manufacturing lead time

#### 4. Shipping Performance
- **Average shipping time:** 5.28 days
- Carrier-wise shipping time analysis (Carrier A, B, C)

#### 5. Inventory Management
- Stock levels by SKU visualization

#### 6. Profit Gauge
- Visual profit indicator showing $92.04K
- Range scale from 0.00K to 300.00K

## Data Model

The dashboard utilizes the following key data dimensions:

### Available Data Fields
- Customer demographics
- Defect Rate %
- Discount rates
- Lead time & Lead times
- Location
- Manufacturing costs & lead time
- Number of products sold
- Order quantities
- Price
- Product type
- Production volumes
- Profit metrics (Profit, Profit Margin %, Profit Max, Profit Target)
- Revenue (Revenue generated, Revenue Target)
- Routes
- Shipping carriers, costs, and times
- SKU
- Stock levels
- Supplier name
- Total Costs & Total Orders & Total Revenue
- Transportation modes

## Technical Requirements

### Software
- Microsoft Power BI Desktop (Latest version recommended)
- Excel (for data source management)

### Data Sources
- Supply chain transaction data
- Supplier performance data
- Inventory management systems
- Shipping and logistics data

## Getting Started

1. Clone this repository
2. Open the Power BI dashboard file (.pbix)
3. Connect to your data sources
4. Refresh the data to see updated metrics

## Usage

This dashboard is designed for:
- Supply chain managers monitoring operational performance
- Executives tracking profitability and efficiency metrics
- Logistics teams optimizing shipping and carrier performance
- Procurement teams evaluating supplier performance
- Inventory managers tracking stock levels
